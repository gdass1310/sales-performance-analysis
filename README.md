# sales-performance-analysis
An Excel-based Sales Analysis Dashboard project focused on analyzing Vrinda Store's sales data. This project includes data cleaning, exploratory analysis, Pivot Tables, Pivot Charts, and interactive dashboards to identify sales trends, customer behaviour, top-performing states, and business insights for data-driven decision making.
Project Highlights:
Cleaned and transformed raw sales data for analysis.
Created interactive dashboards using Pivot Tables, Pivot Charts, and Slicers.
Analyzed sales trends, order status, customer demographics, and sales channels.
Identified top-performing states, customer segments, and key business insights.
Skills Demonstrated:

Excel | Data Cleaning | Data Analysis | Pivot Tables | Dashboard Creation | Data Visualization
