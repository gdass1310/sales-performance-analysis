# 📊 Vrinda Store Sales Analysis Dashboard (Excel)

## Project Overview

This project focuses on analyzing Vrinda Store's sales data to uncover meaningful business insights related to sales performance, customer behaviour, order trends, and revenue contribution.

The objective of this project was to transform raw sales data into an interactive Excel dashboard that helps in understanding key performance indicators (KPIs), identifying growth opportunities, and supporting data-driven business decisions.

---

## Business Problem

Vrinda Store wanted to understand its overall sales performance and customer purchasing patterns.

The analysis aimed to answer key business questions:

- Which customer segments contribute the most to sales?
- Which states generate the highest revenue?
- Which sales channels perform better?
- What is the order delivery and cancellation pattern?
- How do customer demographics impact purchasing behaviour?

# Dataset Description

The dataset contains order-level sales information including:

- Order ID
- Customer Details
- Gender
- Age Group
- Product Information
- Sales Amount
- Order Status
- Sales Channel
- Location Details

The dataset was analyzed to understand sales trends, customer preferences, and business performance.

# Project Workflow

## 1. Data Cleaning & Preparation

Performed data preprocessing to make the dataset analysis-ready:

- Removed duplicate records
- Standardized data formats
- Organized raw data into structured tables
- Checked data consistency
- Prepared data for analysis

---

## 2. Exploratory Data Analysis (EDA)

Analyzed the dataset to identify:

### Sales Performance Analysis

- Overall sales trends
- Revenue contribution
- Monthly sales performance
- Order volume patterns

### Customer Analysis

- Gender-wise sales contribution
- Age group purchasing behaviour
- Customer segmentation insights

### Geographic Analysis

- State-wise sales performance
- Identification of top-performing regions

### Order Analysis

- Delivered orders
- Cancelled orders
- Returned orders

### Sales Channel Analysis

Compared performance across different sales channels to understand channel effectiveness.

# 📊 Dashboard Features

Created an interactive Excel dashboard containing:

- Sales KPIs
- Customer analysis
- Order status analysis
- State-wise sales performance
- Sales channel comparison
- Age and gender analysis

The dashboard uses Pivot Tables, Pivot Charts, and Slicers to allow users to interactively explore business performance.

# 💡 Key Insights

The analysis helped identify:

- Customer purchasing patterns across different demographics.
- High-performing states contributing significantly to revenue.
- Sales channels driving maximum business value.
